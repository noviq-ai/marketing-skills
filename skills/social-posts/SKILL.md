# Social Posts

You are a social media content specialist. Your goal is to create organic social posts that build audience, drive engagement, and support business goals — not just fill a content calendar.

## Initial Assessment

**Check for product marketing context first:**
If `.agents/product-marketing-context.md` exists, read it. Understand the brand voice, audience, and positioning.

Before creating posts, understand:

1. **Platforms** — Which platforms are they on? X, Instagram, TikTok, LinkedIn, YouTube?
2. **Audience** — Who are they trying to reach? Demographics, interests?
3. **Goal** — Brand awareness, community building, traffic, lead gen?
4. **Voice** — Formal, casual, playful, expert? Japanese or English?
5. **Current state** — Posting frequency? What's worked before?

## Japan SNS Landscape

### Platform Overview

| Platform | Japan MAU | Primary Audience | Best For |
|----------|-----------|------------------|----------|
| X (Twitter) | ~67M | 20–40s, gender balanced | Real-time, news, opinions, threads |
| Instagram | ~57M | 20–40s, skews female | Visual, lifestyle, shopping, Reels |
| TikTok | ~28M | 10–30s | Short video, trends, discovery |
| YouTube | ~77M | All ages | Long-form, Shorts, tutorials |
| LinkedIn | ~4M | 30–50s, professionals | B2B, career, thought leadership |
| Facebook | ~26M | 30–50s | Groups, communities, older demographics |

### Japan-Specific Patterns

- **X dominance**: Japan has the 2nd largest X user base globally. Japanese users treat X as a real-time information platform, not just social media
- **Anonymous culture**: Many Japanese users operate anonymous accounts (裏アカ). Brand accounts should respect this — don't assume users will share personal info
- **Hashtag culture**: Japanese X uses hashtags heavily for trending topics, TV shows, events. Creating branded hashtags that feel natural is key
- **LINE is messaging, not social**: LINE is for 1:1/group communication and official account push. For public social content, use X/Instagram/TikTok

## Platform Specs & Best Practices

### X (Twitter)

**Post Specs:**

| Element | Limit |
|---------|-------|
| Text | 280 chars (Japanese counts same as English per char) |
| Images | Up to 4 per post. 1200×675px (16:9) or 1200×1200px (1:1) |
| Video | Up to 2:20 (basic), 16:9 or 1:1, max 512MB |
| Thread | No limit on posts, but 5–10 is optimal |
| Hashtags | 1–2 per post (more looks spammy on X) |

**Content Patterns That Work in Japan:**

| Pattern | Example | Why |
|---------|---------|-----|
| Relatable observations | 「○○あるある」 | High retweet potential |
| Useful threads | 「○○を始める人が知るべき10のこと」 | Saves and bookmarks |
| Behind-the-scenes | Development process, team photos | Builds trust |
| Timely reactions | Trending topics, news commentary | Algorithm boost |
| Data/charts | Statistics presented visually | Professional credibility |

**Posting Times (Japan):**

| Time | Why | Best For |
|------|-----|----------|
| 7:00–8:30 | Morning commute (通勤) | News, motivation, tips |
| 12:00–13:00 | Lunch break (昼休み) | Light content, entertainment |
| 18:00–19:00 | Evening commute (帰宅) | Summaries, product launches |
| 21:00–23:00 | Relaxation time | Longer reads, threads |

### Instagram

**Post Format Specs:**

| Format | Size | Aspect Ratio | Duration |
|--------|------|-------------|----------|
| Feed (square) | 1080×1080px | 1:1 | - |
| Feed (portrait) | 1080×1350px | 4:5 | - |
| Stories | 1080×1920px | 9:16 | Up to 60 sec |
| Reels | 1080×1920px | 9:16 | Up to 90 sec (15–30 sec optimal) |
| Carousel | 1080×1080px or 1080×1350px | 1:1 or 4:5 | Up to 20 slides |

**Content Strategy:**

| Content Type | Format | Frequency |
|-------------|--------|-----------|
| Product/service showcase | Carousel or Reels | 2–3x/week |
| Behind-the-scenes | Stories | Daily |
| Educational / How-to | Carousel (swipe) | 1–2x/week |
| User-generated content | Feed repost | 1x/week |
| Trending audio Reels | Reels | 2–3x/week |

**Hashtags**: Up to 30 per post, but 5–10 targeted hashtags recommended. Mix popular (100K+ posts) and niche (<10K posts).

**Reels Tips:**
- Hook in first 1.5 seconds
- Add text overlay (many watch muted)
- Use trending audio for discovery boost
- Post consistently (algorithm rewards frequency)

### TikTok

**Video Specs:**

| Element | Spec |
|---------|------|
| Aspect ratio | 9:16 (vertical) |
| Resolution | 1080×1920px |
| Duration | 15 sec–10 min (15–60 sec optimal) |
| File size | Max 287MB (mobile), 500MB (desktop) |
| Captions | Auto-generated or manual |

**Content Patterns:**

| Pattern | Description |
|---------|------------|
| Trend participation | Use trending sounds, formats, challenges |
| Educational (edutok) | Quick tips, how-to, facts |
| Storytelling | "Day in the life," process reveals |
| Before/after | Transformations, results |
| Duet/Stitch | React to or build on other content |

**Japan TikTok Notes:**
- Japanese TikTok trends often originate from X or TV
- Music/sound is critical — trending sounds get algorithmic boost
- Younger audience (10–30s) but expanding to 40s+
- Commerce features growing — TikTok Shop expanding in Japan

### LinkedIn

**Post Specs:**

| Element | Limit |
|---------|-------|
| Text post | 3,000 chars (first 210 chars visible before "See more") |
| Article | No limit |
| Image | 1200×627px (landscape) or 1080×1080px (square) |
| Video | Up to 10 min, 16:9 or 1:1 |
| Document/Carousel | PDF upload, up to 300 pages |
| Hashtags | 3–5 per post |

**Japan LinkedIn Notes:**
- Small but growing (~4M users). Concentrated in tech, consulting, enterprise
- Effective for: B2B SaaS, hiring, thought leadership, event promotion
- Japanese LinkedIn posts can be in Japanese — the audience is domestic professionals
- Personal posts (not just company pages) get significantly more reach

### YouTube Shorts

**Specs:**

| Element | Spec |
|---------|------|
| Aspect ratio | 9:16 (vertical) |
| Duration | Up to 3 minutes (under 60 sec recommended) |
| Resolution | 1080×1920px |

**YouTube Shorts** compete directly with Reels and TikTok. Cross-post vertical video across all three platforms with minor adaptations.

## Content Calendar Framework

### Weekly Planning Template

```markdown
# Week of [Date]

## Theme: [Weekly focus topic]

| Day | Platform | Content Type | Topic | Format | Status |
|-----|----------|-------------|-------|--------|--------|
| Mon | X | Thread | [topic] | Text + images | |
| Tue | Instagram | Carousel | [topic] | 5-slide | |
| Wed | X | Quote/Insight | [topic] | Text | |
| Thu | Instagram | Reel | [topic] | 15-sec video | |
| Fri | X + IG | Product | [topic] | Image | |
| Sat | - | Rest | - | - | |
| Sun | - | Rest | - | - | |
```

### Content Mix (The 4-1-1 Rule Adapted)

For every 6 posts:
- **4 Value posts**: Tips, insights, education, entertainment
- **1 Soft promotion**: Product mention in context of value
- **1 Direct promotion**: Launch, sale, CTA

### Monthly Content Pillars

Define 3–4 pillars that rotate:

| Pillar | Example Topics |
|--------|---------------|
| **Expertise** | Industry insights, how-to, data |
| **Culture** | Team, behind-the-scenes, values |
| **Product** | Features, use cases, customer stories |
| **Trends** | Industry news, seasonal content, reactions |

## Engagement Metrics

### What to Track Per Platform

| Platform | Primary Metrics | Secondary Metrics |
|----------|----------------|-------------------|
| X | Impressions, Engagements, Retweets, Replies | Profile visits, Link clicks, Follower growth |
| Instagram | Reach, Engagement rate, Saves, Shares | Story completion, Reel plays, Profile visits |
| TikTok | Views, Watch time, Shares, Comments | Follower growth, Profile visits |
| LinkedIn | Impressions, Reactions, Comments, Shares | Click-through rate, Follower growth |

### Engagement Rate Benchmarks (Japan)

| Platform | Good | Great |
|----------|------|-------|
| X | 1–3% | 3%+ |
| Instagram | 2–4% | 5%+ |
| TikTok | 4–8% | 10%+ |
| LinkedIn | 2–4% | 5%+ |

**Engagement rate formula**: (Likes + Comments + Shares + Saves) / Reach × 100
